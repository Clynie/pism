Standard names of NetCDF variables, related to ice sheet modeling, including PISM current and proposed



Existing names in CF 1.4 Standard Name Table

These names are a small subset of the CF 1.4 standard name table. The list here is just (i) those with "land_ice" in the name and (ii) those relevant to the Parallel Ice Sheet Model (PISM). It is in alphabetical order.

Some of the short names are shared with GLIMMER and CISM. The names below starting with "land_ice" are believed to have all been submitted by Magnus Hagdorn to the CF committee circa 2003 (date?).

Like the legendary Eskimo, the CF standard name table already has a bazillion names for snow. No attempt at using those names is made in PISM.

Readers other than PISM programmers should ignor the final column. PAC=PISMAtmosphereCoupler and POC=PISMOceanCoupler. Note PISM also uses "grid_mapping_name" (in only one case; the name is "polar_stereographic"), but that is not technically a standard_name. Because PISM uses UDUNITS, PISM will write some of these in "glaciological units", for instance m a-1 instead of m s-1, but this is allowed under CF and is handled by UDUNITS.
